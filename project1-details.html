イントロダクション

このプロジェクトは、飲食業界で25年間の経験を積んできた私が、全くのIT未経験から新たな挑戦として始めた学習の成果をまとめたものです。

現在、キャルITカレッジ大阪校で6ヶ月間のJavaプログラミング学習を続ける中で、PHPやAWS、クラウド技術にも興味を持ち、日々学びを深めています。

このウェブサイトは、私が学習の過程で培ったスキルを実践に活かし、HTML、CSS、JavaScript、PHPを用いて作成したものです。このプロジェクトを通じて、IT業界への理解を深めるとともに、実践的なスキルを身につけることを目指しています。

プロジェクトの詳細

このホームページを制作した背景

キャルITカレッジ大阪校でJavaを学びつつ、手を動かす楽しさを知るためにこのウェブサイト制作をスタートしました。

HTML/CSSで見た目を整え、JavaScriptで動きを加え、PHPで仕組みを作る過程は、まるで新しいレシピを開発するようでワクワクが止まりません。

クラウド環境（AWS）での制作は、サーバーの設定ひとつとっても「なぜ？」を追求する姿勢が大切だと実感しました。

使用した技術スタック

  • HTML5 - セマンティックなマークアップ、アクセシビリティ対応
  • CSS3 - レスポンシブデザイン、CSS変数、アニメーション
  • JavaScript (ES6+) - 非同期処理、Fetch API、イベントハンドリング
  • PHP - お問い合わせフォームのバックエンド処理、PHPMailer
  • AWS - API Gateway、Lambda、SES（Simple Email Service）

日々Javaと向き合いながらも、いろいろなことをやりたい欲張りな性格なので、様々なことを学ぶことのできる日々がとても充実していて、毎日楽しく過ごしています。

Javaの進捗は、オブジェクト指向や例外処理などを中心に理解を深めている最中です。現在のスキルはまだまだですが、基本をしっかりと固める時期だと思って取り組んでいます。

個人的な予習として、現在はPHPの学習と、JavaでWebアプリを作成するためにSpringフレームワークを中心に自習しています。
