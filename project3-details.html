🚀 開発予定のアプリケーション

計画中

タスク管理アプリ

概要: ユーザーがタスクを登録・管理できるWebアプリケーション

主な機能:

  • ユーザー認証・ログイン機能
  • タスクの作成・編集・削除（CRUD操作）
  • 優先度設定と期限管理
  • カテゴリー別のタスク分類
  • 完了状態の管理

使用技術: Spring Boot, Thymeleaf, MySQL, Bootstrap

構想中

書籍管理システム

概要: 個人の読書記録を管理し、読書体験を記録できるアプリケーション

主な機能:

  • 書籍情報の登録と検索
  • 読書状態の管理（読了・読書中・未読）
  • 評価とレビューの記録
  • 読書統計の可視化
  • おすすめ書籍のリスト化

使用技術: Spring Boot, Spring Data JPA, MySQL, RESTful API
