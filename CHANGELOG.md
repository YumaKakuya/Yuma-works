# 更新履歴（Changelog）

このファイルには、ポートフォリオサイトの主要な変更履歴を記録します。

形式は [Keep a Changelog](https://keepachangelog.com/ja/1.0.0/) に基づいています。

## [2.0.0] - 2025-02-05

### 追加 (Added)
- **SEO対策**: メタディスクリプション、OGPタグ、Twitter Cardの追加
- **アクセシビリティ**: ARIA属性、role属性を全ページに実装
- **セキュリティ機能**: 
  - レート制限機能（5分以内3回まで）
  - CSRF保護
  - 入力バリデーション強化
  - エラーログ記録
- **パフォーマンス最適化**:
  - 画像の遅延読み込み（lazy loading）
  - フォントのpreconnect設定
  - width/height属性の追加
- **プロジェクト詳細ページの大幅改善**:
  - project1-details.html: タイムライン、技術スタック、今後の計画
  - project2-details.html: GitHub APIリポジトリ表示、学習フォーカス
  - project3-details.html: 学習中の技術、開発予定アプリ、学習目標
- **新規ドキュメント**:
  - README.md（プロジェクト概要、セットアップ手順）
  - DEPLOY.md（デプロイガイド）
  - CHANGELOG.md（このファイル）
  - .env.example（環境変数テンプレート）
  - .gitignore（Git除外設定）
  - composer.json（PHP依存関係管理）
  - robots.txt（検索エンジン向け）
  - sitemap.xml（サイトマップ）

### 変更 (Changed)
- **フォルダ名**: `image` → `images` に統一
- **画像パス**: 全HTMLファイルで実際の画像ファイル名に修正
- **ナビゲーション**: Portfolio1.htmlへのリンクに統一
- **SNSリンク**: GitHubのみに整理（YumaKakuyaアカウント）
- **フォームUI**: より使いやすく、アクセシブルなデザインに改善
- **CSS**: 
  - 詳細ページ用スタイルの大幅追加
  - フォームスタイルの改善
  - カード型レイアウトの導入
  - アニメーション効果の追加
- **JavaScript**:
  - async/await構文に更新
  - エラーハンドリングの強化
  - タイムアウト設定の追加
  - メッセージ表示UIの改善
- **sendmail.php**:
  - 環境変数対応
  - セキュリティ機能の大幅強化
  - エラーログ記録の追加
  - HTML形式メールの改善

### 修正 (Fixed)
- CSSパスの不整合を解消
- 画像ファイル名の不一致を修正
- フォームバリデーションの改善
- 全ページでのレスポンシブ対応の統一
